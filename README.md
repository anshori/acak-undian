# ACAK UNDIAN DOORPRIZE
Acak undian doorprize untuk memeriahkan HUT RI ke-79

Acak undian ini diatur angka yang keluar secara _default_ 001 - 079 dengan jumlah doorprize 24 buah.

Silahkan sesuaikan jumlah doorprize dan jumlah hadirin di kolom input **Jumlah Doorprize** dan **Jumlah Hadirin**.

![Screenshot](assets/image/Screenshot.png)


#### Catatan
- Angka yang sudah keluar dan dimunculkan di kotak **Angka Sudah Keluar** tidak akan keluar lagi
- Jika jumlah doorprize sama dengan jumlah angka yang keluar, maka akan muncul pesan _**"Selesai. Semua nomer undian sudah keluar. Acak undian ini akan direset ulang"**_
- Untuk mereset ulang acak undian ini, gunakan tombol _**shift + klik refresh browser**_
